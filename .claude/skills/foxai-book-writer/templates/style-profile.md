# STYLE PROFILE — phong cách viết của sách (cần user duyệt)

> Cách lập: hỏi user 5 mục dưới; nếu user đưa **đoạn văn mẫu** (từ sách họ thích hoặc chính họ viết),
> phân tích mẫu để điền thay vì bắt user tự mô tả. Mẫu thật > mô tả trừu tượng.

## 1. Giọng & vai người viết
- **Xưng hô:** (tôi—bạn / chúng ta / trung tính "người đọc")
- **Vai:** (người đi trước kể lại / chuyên gia hướng dẫn / đồng nghiệp trao đổi)
- **Nhiệt độ:** (thẳng và khô / ấm có ví dụ đời thường / truyền cảm hứng)

## 2. Nhịp câu & đoạn
- Độ dài câu trung bình (ngắn gọn 10–15 từ / trung bình / câu dài nhiều mệnh đề)
- Độ dài đoạn (2–4 câu / 5–7 câu) · Mật độ heading (## mỗi ~X đoạn)
- Mở chương bằng: (câu chuyện / câu hỏi / khẳng định mạnh) · Kết chương bằng: (tóm tắt / cầu nối sang chương sau / call-to-action)

## 3. Thuật ngữ & ví dụ
- Thuật ngữ kỹ thuật: giữ tiếng Anh + giải thích Việt lần đầu (mặc định FOXAI)
- Ví dụ ưu tiên bối cảnh: (Việt Nam / quốc tế / ngành cụ thể của độc giả)
- Mật độ callout: (nhiều — mỗi section 1 hộp / vừa / tối giản)

## 4. Đoạn văn mẫu do user cung cấp (QUAN TRỌNG NHẤT)
> Dán 1–3 đoạn mẫu user đưa vào đây, kèm nhận xét đã trích xuất được gì từ mẫu
> (nhịp câu, cách dùng ẩn dụ, mức độ trực tiếp...). Khi viết mỗi chương, đọc lại mục này trước.

```
[đoạn mẫu 1]
```
Phân tích: ...
